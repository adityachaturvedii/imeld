Background & Aim

Current non-invasive tests (NITs) for predicting hemodynamic response to beta-blockers lack the precision required for personalized therapy. We aimed to develop a robust, "high-confidence" prediction model using advanced machine learning techniques, specifically addressing the challenge of class imbalance in rebleed events.

Methodological Innovation (The Novelty)

  • SMOTE Enhancement: Unlike standard logistic regression, we employed Synthetic Minority Over-sampling Technique (SMOTE) to boost sensitivity for the rare rebleed class (17.7%), achieving a 5.5-fold risk gradient separation.
  • 7-Tier Feature Engineering: We derived 107 novel features, identifying a critical pharmacodynamic interaction: Dose × SSM_response, which outperformed individual metrics.
  • Rigorous Validation: Version 2.0 analysis included Bootstrap 95% Confidence Intervals (1000 iterations) and Decision Curve Analysis (DCA) to quantify net clinical benefit.

Conclusion

The AI-augmented model achieves an AUC of 0.86 (95% CI: 0.80-0.91) for HVPG response. By defining high-confidence zones, the model can safely spare 40% of patients from invasive HVPG monitoring while maintaining >92% specificity, offering a definitive precision medicine tool for portal hypertension.
